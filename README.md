# HealthMate
HealthMate, kullanıcıların günlük beslenme alışkanlıklarını ve su tüketimlerini kayıt altına alarak sağlıklı yaşam hedeflerine ulaşmalarını sağlayan bir dijital asistandır.
İlgili database autharization kısımları oluşturuldu. 
Firebase Bağlantısı sağlanıldı.
Gerekli sayfalar oluşturuldu. 
Kullanıcı girişi istenilen şekilde sağlanıldı.
Sayfaların backend ve ui kısımları yazılmaya başlanıldı, yeni özellikler eklendikçe güncellenecek.
